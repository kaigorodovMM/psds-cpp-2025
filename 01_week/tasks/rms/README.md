# Вычисление RMS

Необходимо написать функцию `CalculateRMS`, которая 
принимает массив вещественных чисел и размер массива, а возвращает истинное 
среднеквадратическое значение ([Root Mean Square]). Вычисление производится 
на одном окне, равном длине всего массива.

В случае пустого массива вернуть `0.0`.

## Ограничения

- Нельзя использовать библиотечные функции для вычисления RMS.

## Примечание

- Вычисление квадратного корня можно осуществить функцией `sqrt` из `<cmath>`
