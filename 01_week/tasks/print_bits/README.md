# Битовое представление числа

Необходимо написать функцию `PrintBits`, которая принимает целое число и
размер в байтах и выводит на экран битовое представление числа в формате 
`0bXXXX'XXXX`. Размер нужен, чтобы не писать отдельно функцию для каждого
типа.

В конце вывода необходим перевод строки.

При значении размера байт $bytes \notin (0, 8]$ функция ничего не выводит.

## Ограничения

- Запрещено использовать `std::bitset`.
