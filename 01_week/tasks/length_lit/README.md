# Пользовательские литералы длины

Необходимо написать пользовательские литералы для преобразования мер длины.
Литералы должны осуществлять преобразования между футами ([foot]), дюймами
([inch]), сантиметрами и метрами.

Формат литерала: `_{1}_to_{2}`, где вместо `{1}` и `{2}` используются
`ft`, `in`, `cm`, `m` для соответствующих величин.

Все преобразования возвращают тип `double`.
