# Решение квадратного уравнения

Реализуйте функцию `SolveQuadratic`, которая решает [квадратное уравнения]
вида $(a{\cdot}x^2 + b{\cdot}x + c)$. Коэффициенты для простоты имеют
целочисленный тип. Решение выводится в стандартный поток в определенном
формате:

- `x1 x2` (причем `x1 < x2`) - когда два корня.
- `x1` - когда один корень. 
- `no solutions` - когда нет решений.
- `infinite solutions` - когда бесконечное количество решений.

Точность вывода в поток необходимо задать 6 символов.

## Примечание

- Рекомендуется рассмотреть все особые случаи значения коэффициентов.
- Используйте `std::setprecision(6)` из заголовочного файла `<iomanip>`
